# AI-Upskilling-App

A lightweight, no-build web app for your CAIO upskilling journey.

## Features
- CAIO-aligned roadmap modules with completion tracking and notes
- Security coverage indicator
- Executive alignment checklist (CEO, CISO, IT Apps, Business Units)
- Adoption metrics tracking
- AI Radar briefs with security-first recommendations
- Local snapshot export/import for portability

## Run locally
1. Open `index.html` directly in a browser, or
2. Serve the folder with a simple static server.

Example with Python:

```bash
python -m http.server 8080
```

Then open `http://localhost:8080`.

## Deployment
This is static and can be deployed to any static host (GitHub Pages, Netlify, Vercel static, or your own domain `caioskills.app`).
